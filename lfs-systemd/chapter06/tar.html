6.67. Tar-1.29

Tar 软件包包含一个归档程序。

大概编译时间: 3.2 SBU
所需磁盘空间: 40 MB

6.67.1. 安装 Tar

准备编译 Tar： 

FORCE_UNSAFE_CONFIGURE=1  \
./configure --prefix=/usr \
            --bindir=/bin

配置选项的含义：

FORCE_UNSAFE_CONFIGURE=1

强制以 root 用户运行 mknod 的测试。通常认为以 root 用户运行该测试是危险的，但由于是在部分构建的系统上运行，这样并没有问题。

编译软件包： 

make

用以下命令测试结果(大概 1 SBU)： 

make check

安装软件包： 

make install
make -C doc install-html docdir=/usr/share/doc/tar-1.29

6.67.2. Tar 软件包内容

安装的程序: tar
安装的目录: /usr/share/doc/tar-1.29

简要介绍

tar

创建归档文件，从归档文件提取文件，列出归档文件的内容，归档文件也被成为 tar 包
