6.53. Gawk-4.1.4

Gawk 软件包包含用于操作文本文件的程序。

大概编译时间: 0.3 SBU
所需磁盘空间: 36 MB

6.53.1. 安装 Gawk

准备编译 Gawk： 

./configure --prefix=/usr

编译软件包： 

make

用以下命令测试结果： 

make check

安装软件包： 

make install

如果需要的话，安装帮助文档： 

mkdir -v /usr/share/doc/gawk-4.1.4
cp    -v doc/{awkforai.txt,*.{eps,pdf,jpg}} /usr/share/doc/gawk-4.1.4

6.53.2. Gawk 软件包内容

安装的程序: awk (链接到 gawk), gawk, gawk-4.1.4, 和 igawk
安装的库: filefuncs.so, fnmatch.so, fork.so, inplace.so, ordchr.so, readdir.so, readfile.so, revoutput.so, revtwoway.so, rwarray.so, testext.so, 和 time.so
安装的目录: /usr/lib/gawk, /usr/libexec/awk, /usr/share/awk, 和 /usr/share/doc/gawk-4.1.4

awk

gawk

gawk-4.1.4
