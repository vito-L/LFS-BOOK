6.24. Sed-4.4

Sed 软件包包含一个流编辑器。

大概编译时间: 0.3 SBU
所需磁盘空间: 25 MB

6.24.1. 安装 Sed

首先修复 LFS 环境中的问题，然后移除一个失败的测试： 

sed -i 's/usr/tools/'       build-aux/help2man
sed -i 's/panic-tests.sh//' Makefile.in

准备编译 Sed： 

./configure --prefix=/usr --bindir=/bin

编译软件包并生成 HTML 文档： 

make
make html

输入命令查看结果： 

make check

安装软件包和它的文档： 

make install
install -d -m755           /usr/share/doc/sed-4.4
install -m644 doc/sed.html /usr/share/doc/sed-4.4

6.24.2. Sed 软件包内容

安装的程序: sed
安装目录: /usr/share/doc/sed-4.4

简要介绍

sed

过滤器，一次性转换文本文件
