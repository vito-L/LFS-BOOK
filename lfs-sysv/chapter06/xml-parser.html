6.41. XML::Parser-2.44

XML::Parser 模块是到 James Clark 的 XML 解析器的 Perl Expat 接口。

大概编译时间: 少于 0.1 SBU
所需磁盘空间: 2.0 MB

6.41.1. 安装 XML::Parser

准备编译 XML::Parser： 

perl Makefile.PL

编译软件包： 

make

用以下命令测试结果： 

make test

安装软件包： 

make install
