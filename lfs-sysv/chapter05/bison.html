5.17. Bison-3.0.4

Bison 软件包包含一个语法生成器。

大概编译时间: 0.3 SBU
所需磁盘空间: 32 MB

5.17.1. 安装 Bison

配置 Bison 准备编译： 

./configure --prefix=/tools

编译软件包： 

make

为了测试结果，输入： 

make check

安装软件包： 

make install

该软件包的详细信息参见 第 6.29.2 节 “Bison 软件包内容.”。
